# 📦 Git Stash Quick Guide

## 🧠 What is Stash?
Temporarily save uncommitted changes without committing.

---

## ✅ Save Changes
```bash
git stash

📂 List Stashes
git stash list

🔄 Apply Last Stash
git stash apply

❌ Drop Stash
git stash drop

🚀 Tip
Use stash when switching branches without committing.
### Branch
```
