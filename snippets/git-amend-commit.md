# ✏️ Amend Last Git Commit

## 🧠 Use Case
Fix last commit message or add missed changes.

---

## ✅ Change Commit Message
```bash
git commit --amend -m "New commit message"

➕ Add Changes to Last Commit

git add .
git commit --amend --no-edit

⚠️ Warning

Avoid amending commits that are already pushed to shared repos.

🚀 Tip

Great for quick fixes before pushing.
```
