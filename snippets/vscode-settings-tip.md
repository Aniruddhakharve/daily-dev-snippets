# VS Code Settings & Tips

1. Auto-save on focus change:
   - Settings (Ctrl + ,) → Search "files.autoSave" → set to "onFocusChange"

2. Format on save (with Prettier/ESLint):
   - Install Prettier extension
   - Settings → "editor.formatOnSave": true

3. Quick snippet for Python main:
   Type `ifname` → Tab → auto "if __name__ == '__main__':"

Pro tip: Use "Settings Sync" extension to backup across machines.
