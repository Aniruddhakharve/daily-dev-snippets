# VS Code Shortcuts (Daily Use)

1. Quick open file: Ctrl + P (Cmd + P on Mac)
2. Toggle terminal: Ctrl + ` (backtick)
3. Multi-cursor edit: Alt + Click (or Ctrl + Alt + Up/Down)
4. Format document: Shift + Alt + F
5. Comment line: Ctrl + /

Pro tip: Install "Prettier" extension for auto-format on save.
