# 🚫 .gitignore Best Practices

## 🔥 Common Entries

node_modules/
.env
*.log
dist/
build/


---

## 🧠 Tips
- Always ignore secrets (.env)
- Avoid committing build artifacts
- Use global gitignore if needed

---

## 🚀 Pro Tip
Use:
```bash
git rm -r --cached .

to apply new gitignore rules.
```
